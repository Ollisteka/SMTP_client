Отправьте емэйл в любую точку мира, с помощью этого чудесного клиента, работающего по протоколу SMTP!
---
HOW TO:
1) Отредактируйте файл input.json:

     "From": ваш адрес (на яндексе)
       "To": список адресов получателей
     "Text": путь к файлу с текстом емэйла
  "Subject": тема сообщения
  "Attachments": список файлов, которые вы хотите прикрепить к письму

2) Запустите программу:
    python client.py

3) Profit!
---
Как это работает:
Клиент подключается к SMTP серверу яндекса, проходит авторизацию, формирует заголовки письма, кодирует аттачменты в формате MIME, отправляет письмо.
---
Доп. возможности:
python client.py -c
    Запуск клиента в консольном режиме - напишите свой емэйл в интерактивном режиме!
    Чтобы узнать доступные команды - выполните команду help
    Пример SMTP сессии смотрите на вики (https://ru.wikipedia.org/wiki/SMTP#Пример_простейшей_SMTP-сессии), возможно, когда-нибудь он появится и тут :)

python client.py --help
    Ещё два бонусных параметра :)
